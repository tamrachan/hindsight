# Hindsight
From History to Market Insight.

Team Hindsight for the BlackRock x WICS x FinTech hackathon 14/03/2026 where the theme was to "introduce and educate students to the world of personal finance".

## Project Overview

This project is an interactive financial tracker that shows how different asset classes—such as stocks, bonds, commodities, and cryptocurrencies—react to major global events. Users can explore a timeline of significant news, including wars, financial crises, terrorist attacks, and interest rate changes, and see how markets responded during those periods. 

By selecting specific events, students can visualize and compare the performance of different asset classes, helping them understand how real-world events influence financial markets, investor behavior, and risk across the global economy.
